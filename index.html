What is a CSS Gradient?

A CSS gradient displays smooth transitions between two or more colors. Use gradients as backgrounds for buttons, cards, hero sections, or full websites. All gradients here are 100% free for personal and commercial use.

Frequently Asked Questions

How to use CSS gradient in background?

Copy the CSS code and paste it in your stylesheet: div { background: linear-gradient(...); }

Can I use these gradients for commercial projects?

Yes, all gradients are free for personal and commercial use without attribution.

What's the difference between linear and radial gradient?

Linear gradients go in one direction. Radial gradients spread from a center point outward in a circle.
